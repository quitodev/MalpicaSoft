# MalpicaSoft

MalpicaSoft es un sistema de control de stock que desarrollé (uno de mis primeros proyectos), el cual permite tener un control de las compras y ventas que realiza un comercio.

El proyecto está finalizado pero no está en funcionamiento, debido a que prioricé trabajar en otros proyectos que tenía en paralelo.

## Tecnologías implementadas
- JAVA
- MySQL
